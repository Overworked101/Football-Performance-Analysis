# UCL Barcelona vs Newcastle – Football Performance Analysis

## 📌 Overview
This project presents a data-driven analysis of a UEFA Champions League match between FC Barcelona and Newcastle United. The objective was to analyze shot distribution and derive tactical insights using Python.

---

## ⚙️ Methodology

### Data Collection
- Match data and shot locations were manually extracted from SofaScore

### Data Processing
- Structured dataset created in CSV format (`shots.csv`)

### Data Visualization
- Shot map generated using Python:
  - Pandas
  - Matplotlib

---

## 📊 Key Analysis

- Shot distribution analysis (location-based)
- Comparison of attacking patterns
- Identification of high-probability scoring zones

---

## 🧠 Key Insights

- Barcelona generated more central, high-quality chances through structured build-up play  
- Newcastle relied more on transitions, leading to dispersed shot patterns  
- Most goals originated from close-range positions, highlighting the importance of shot location
